## Vim 常用指令

### 模式切換
| 指令 | 說明 |
|------|------|
| `i` | 在游標前進入插入模式 |
| `a` | 在游標後進入插入模式 |
| `o` | 在下一行新增並進入插入模式 |
| `O` | 在上一行新增並進入插入模式 |
| `v` | 進入視覺模式（字元選取） |
| `V` | 進入視覺模式（整行選取） |
| `Ctrl+v` | 進入區塊視覺模式 |
| `Esc` | 回到普通模式 |

### 移動游標
| 指令 | 說明 |
|------|------|
| `h / j / k / l` | 左 / 下 / 上 / 右 |
| `w` | 跳到下個單字開頭 |
| `b` | 跳到上個單字開頭 |
| `e` | 跳到單字結尾 |
| `0` | 跳到行首 |
| `^` | 跳到行首第一個非空白字元 |
| `$` | 跳到行尾 |
| `gg` | 跳到檔案開頭 |
| `G` | 跳到檔案結尾 |
| `{數字}G` | 跳到指定行號 |
| `Ctrl+f/b` | 向下/上翻頁 |

### 編輯
| 指令 | 說明 |
|------|------|
| `x` | 刪除游標所在字元 |
| `dd` | 刪除整行 |
| `{n}dd` | 刪除 n 行 |
| `dw` | 刪除到下個單字 |
| `d$` | 刪除到行尾 |
| `yy` | 複製整行 |
| `{n}yy` | 複製 n 行 |
| `yw` | 複製單字 |
| `p` | 貼上（游標後） |
| `P` | 貼上（游標前） |
| `u` | 復原（Undo） |
| `Ctrl+r` | 重做（Redo） |
| `r{字元}` | 取代游標字元 |
| `cc` | 清除整行並進入插入模式 |
| `cw` | 清除到下個單字並進入插入模式 |
| `.` | 重複上一個指令 |

### 搜尋與取代
| 指令 | 說明 |
|------|------|
| `/{關鍵字}` | 向下搜尋 |
| `?{關鍵字}` | 向上搜尋 |
| `n / N` | 下一個 / 上一個搜尋結果 |
| `:%s/舊/新/g` | 全文取代 |
| `:%s/舊/新/gc` | 全文取代（每次確認） |
| `:n,ms/舊/新/g` | 指定行範圍取代 |

### 存檔與離開
| 指令 | 說明 |
|------|------|
| `:w` | 存檔 |
| `:q` | 離開 |
| `:wq` 或 `ZZ` | 存檔並離開 |
| `:q!` | 強制離開（不存檔） |
| `:wq!` | 強制存檔並離開 |
| `:e {檔名}` | 開啟檔案 |

### 多視窗
| 指令 | 說明 |
|------|------|
| `:sp {檔名}` | 水平分割視窗 |
| `:vsp {檔名}` | 垂直分割視窗 |
| `Ctrl+w w` | 切換視窗 |
| `Ctrl+w q` | 關閉視窗 |

### 實用技巧
| 指令 | 說明 |
|------|------|
| `:{數字}` | 跳到指定行號 |
| `%` | 跳到對應的括號 |
| `>>` / `<<` | 縮排 / 反縮排 |
| `={motion}` | 自動排版 |
| `*` | 搜尋游標所在單字 |
| `Ctrl+g` | 顯示目前行號與檔案資訊 |
